# mdrss-cli
## Usage
```bash
CLI tool to generate RSS feed from markdown files using mdrss

Usage: mdrss-cli --input-dir <INPUT_DIR> --output-file <OUTPUT_FILE>

Options:
  -i, --input-dir <INPUT_DIR>      Path to the directory with markdown files
  -o, --output-file <OUTPUT_FILE>  Path to the output rss.xml file
  -h, --help                       Print help
  -V, --version                    Print version
```
## Example
```bash
./mdrss-cli --input-dir <directory_with_md_files> --output-file <path_to_rss.xml>
```
